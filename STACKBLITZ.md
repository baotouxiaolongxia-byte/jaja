# 在线预览方案 - StackBlitz

立即在浏览器中预览网站，无需安装任何东西！

## 步骤：

1. 访问 https://stackblitz.com/
2. 点击 "Import Project"
3. 选择 "Import from GitHub"（需要您先推送到 GitHub）

或者更简单的方式：

## 快速创建在线版本：

1. 访问 https://stackblitz.com/edit/vite-react-ts
2. 删除所有文件
3. 将本地项目的所有文件复制粘贴进去
4. StackBlitz 会自动安装依赖并启动

## 使用 CodeSandbox：

1. 访问 https://codesandbox.io/
2. 点击 "Create Sandbox"
3. 选择 "Import Project"
4. 粘贴您的 GitHub 仓库地址

您的网站将立即在线运行！
